# DODOBOT
Daegu University AI Chatbot Application

대구대학교 인공지능 챗봇 애플리케이션 입니다.
챗봇 기능 외에도 학사일정 교내 건물 위치 등 학교관련 정보를 습득할 수 있는 프로그램 입니다.
